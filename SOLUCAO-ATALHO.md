# Guia Rápido para Seu Amigo - Se o Atalho não funcionar

Se o atalho "Auto-Sync CRM" não funcionar, seu amigo pode:

## OPÇÃO 1: ARQUIVO BAT (MAIS FÁCIL) ✅

1. Na pasta `C:\Users\[seunome]\Desktop\CRM-GitHub`
2. Procure pelo arquivo: **iniciar-auto-sync.bat**
3. **Clique 2X** para abrir
4. Pronto! PowerShell abre e começa o auto-sync

---

## OPÇÃO 2: COMANDO DIRETO

Se quiser fazer manualmente:

```powershell
cd C:\Users\[seunome]\Desktop\CRM-GitHub
.\auto-sync-watch.ps1
```

Mude `[seunome]` para o nome real dele

---

## OPÇÃO 3: CRIAR NOVO ATALHO

Se o atalho original não funcionar, crie um novo:

1. Abra a pasta `C:\Users\[seunome]\Desktop\CRM-GitHub`
2. Clique com botão direito no arquivo **iniciar-auto-sync.bat**
3. Escolha "Enviar para" → "Área de Trabalho (criar atalho)"
4. Pronto! Novo atalho está no Desktop

---

**O MAIS FÁCIL:** Seu amigo pode clicar 2X no `iniciar-auto-sync.bat` dentro da pasta CRM-GitHub
